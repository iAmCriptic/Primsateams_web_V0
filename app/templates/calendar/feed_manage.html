Anleitung zum Abonnieren
Google Calendar:
  1. Öffnen Sie Google Calendar
  2. Klicken Sie auf das "+" neben "Andere Kalender"
  3. Wählen Sie "Von URL"
  4. Fügen Sie den Kalender-Link ein
  5. Klicken Sie auf "Kalender hinzufügen"
Outlook:
  1. Öffnen Sie Outlook
  2. Gehen Sie zu "Kalender" → "Kalender hinzufügen" → "Aus Internet"
  3. Fügen Sie den Kalender-Link ein
  4. Klicken Sie auf "OK"
Apple Calendar (macOS/iOS):
  1. Öffnen Sie die Kalender-App
  2. Gehen Sie zu "Ablage" → "Neuer Kalender-Abonnement"
  3. Fügen Sie den Kalender-Link ein
  4. Klicken Sie auf "Abonnieren"
